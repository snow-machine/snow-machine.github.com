1 版面操作

1.1 标记

1.1.1 一般标记

  • m：放入保留区，g：放入文摘区，*：未读贴子，;：不可回复
  • G=`*+g' M=`*+m' U=`m+;' O=`g+;' b=`m+g'
  • B=`b+*'=`M+g'=`G+m' 8=`b+;'=`O+m'=`U+g' 
[*]---------------------G----------------------[g]
 |- \                       /------------------ |
 |    \                    /                    |
 |      \                 /                     |
 |        \              |                      |
 M         B-------------b-----------8          O
 |                       |             \        |
 |                       |               \      |
 |                       /                 \    |
 |                      /                    \  |
 | -------------------/                       | |
[m]---------------------U----------------------[;]

1.1.2 特殊标记

  • 版面进行收精或合集非标记操作，会使文章带有一些标记，这些标记会与前述的标记 叠加，可以在文章前按`shift+;'，修改标记
  • `X' `%' `#'：普通用户不可见标记，`X'：版主用作标记，用于标记删除 `%'：不可 见的m标记 `#'：仅作标记，不对文章产生任何作用

1.2 管理

1.2.1 版面文章列表

  • `v'：进入信箱
  • `ctrl+p'：发贴，`ctrl+r'：回复到信箱
  • `ctrl+g'：进入文摘区、保留区、同主题模式等
  • `E'：修改文章 `d'：单篇删文，`D'：标记删文，`.'：回收站 `shift+.':自删区 `Y':恢复
  • `ctrl+D'：封人，解封
  • `a'：顺序搜寻作者 `shift+a'：逆序搜寻作者 `ctrl+a'：查看作者
  • `/'：顺序搜寻标题 `shift+a'：逆序搜寻标题
  • `b'：同主题操作，有同主题删除、保留、收精、暂存、制作合集等
  • 'W'：备忘录(进版画面)，模板，治版方针
  • ';`：文章推荐，不可回复，置顶
  • `shift+\'：切分主题
  • `shift+4'：到达最后一篇
  • `shift+6'：溯源定位
  • `shift+7'：找到主题贴

1.2.2 讨论区选单

  • `ctrl+a'：将此讨论区放到个人定制区

1.2.3 阅读文章

  • `r/R'：回复文章 's'(home)：到文章首 `e'(end)：到文章尾
  • `shift+/'：文章内查找
  • `p/->/ctrl+x'：同主题阅读
  • `q/<-/esc'：返回文章列表

1.2.4 编辑界面

  • `pgup/pgdown'：上下翻页
  • `ctrl+pgup/ctrl+pgdown'：到文章首
  • `ctrl+w/x'：写完发表
  • `ctrl+y'：行删除

1.3 精华区

  • `x'：进入精华区
  • `a'：新建文件 `g'：新建文件夹 `d'：删除文件 `s'：看文件名（开关）
  • `i' `J'：放入暂存区，在精华区按`i'，起名加入
  • `m'：移动文件 `x'：剪切文件 `c'：复制文件 `p'：粘贴文件 `S':排序文件

1.3.1 丝路

  • `f'：进入丝路。进入要设置丝路的精华区内，按`f',在丝路上按回车，将此精华区设 为丝路
  • `t/T'：更名 `d/D'：删除 `m/M'：移动 `a'：显示路径
  • 同主题加入丝路，用`b4'
