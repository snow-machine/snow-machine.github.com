1 tomcat远程调试

开发中使用emacs jde2.35，希望使用远程方式调试java代码，从网上看到很多 关于linux下tomcat配置远程调试的方法，都不奏效，仔细查找，终于找到配置 方法： 

在/etc/init.d/Tomcat5.sh中找到：
CATALINA_OPTS="-Djava.library.path=/home/jfclere/jakarta-tomcat-connectors/jni/native/.libs"
改为
CATALINA_OPTS="-Xdebug -Xnoagent -Djava.compiler=NONE 
           -Xrunjdwp:transport=dt_socket,server=y,suspend=n,address=4321"

网上查到的都是： 

CATALINA_OPTS="-server -Xdebug -Xnoagent -Djava.compiler=NONE
           -Xrunjdwp:transport=dt_socket,server=y,suspend=n,address=8788"

多了-server
